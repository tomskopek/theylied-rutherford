# TheyLied.edu

A satire website that applies conspiracy theory aesthetics to the fact that high school teachers knowingly teach the incorrect Rutherford/Bohr model of the atom.

## Features

- Full conspiracy site parody (dramatic quotes, "follow the money", whistleblowers, redacted documents)
- Procedurally generated dark ambient audio drone via Web Audio API (no external files needed)
- Live counter of "students being lied to"
- "Do Your Own Research" links to actual quantum mechanics papers
- Fake newsletter signup
- Responsive design

## Run locally

Just open `index.html` in a browser. No build step, no dependencies.

## Disclaimer

This is satire. Your high school teachers were doing their best. The Bohr model is a useful pedagogical simplification. Please don't yell at Mrs. Henderson.
